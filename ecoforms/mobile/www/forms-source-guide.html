Automático RECOMENDADO

Servidor → Cache → Local

✨ Vantagens

  • Rápido: Usa cache local quando possível
  • Inteligente: Só busca do servidor quando necessário
  • Otimizado: Melhor balanceamento entre velocidade e atualização
  • Econômico: Reduz uso de dados móveis
  • Funciona offline: Usa dados em cache

⚡ Considerações

  • Pode não ter a versão mais recente imediatamente
  • Requer setup inicial com internet

📱 Ideal para

  • Uso geral do sistema
  • Conexão de internet intermitente
  • Trabalho em campo com sinal variável
  • Economia de dados móveis
☁️

Servidor Primeiro SEMPRE ATUALIZADO

Supabase → Cache (fallback)

✨ Vantagens

  • Sempre fresco: Garante a versão mais recente
  • Sincronizado: Dados consistentes com o servidor
  • Atualização automática: Recebe mudanças imediatamente
  • Fallback seguro: Usa cache se servidor indisponível

⚡ Considerações

  • Requer conexão estável à internet
  • Pode ser mais lento (download do servidor)
  • Consome mais dados móveis
  • Dependente de disponibilidade do servidor

📱 Ideal para

  • Escritório com WiFi estável
  • Necessidade de dados mais recentes
  • Trabalho colaborativo intenso
  • Formulários atualizados com frequência
📱

Somente Local OFFLINE

Cache → Local (sem rede)

✨ Vantagens

  • 100% Offline: Funciona sem internet
  • Privacidade: Nenhuma comunicação com servidor
  • Zero dados: Não consome internet
  • Mais rápido: Sem espera por rede
  • Independente: Não depende de servidor

⚡ Considerações

  • Dados desatualizados: Não recebe atualizações
  • Setup inicial obrigatório: Precisa instalar tudo antes
  • Sem novos formulários: Limitado ao que foi instalado
  • Sem sincronização: Mudanças não são compartilhadas

📱 Ideal para

  • Trabalho em áreas sem cobertura de rede
  • Uso em campo remoto
  • Sem plano de dados disponível
  • Necessidade de máxima privacidade
  • Demonstrações offline

💡 Como Escolher?

Tem internet estável? Use Servidor Primeiro para dados sempre atualizados.
Conexão intermitente? Use Automático (recomendado) para melhor experiência.
Trabalha sem internet? Use Somente Local para funcionamento offline completo.

Você pode mudar essa configuração depois nas configurações do aplicativo.
