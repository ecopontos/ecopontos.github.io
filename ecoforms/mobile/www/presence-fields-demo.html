presence-list

PresenceListField

Campo simples e direto com interface de radio buttons em estilo checklist. Ideal para listas pequenas e formulários que precisam de simplicidade.

  • Radio buttons tradicionais
  • Cores por estado (Verde/Vermelho/Cinza)
  • Otimizado para mobile (48px touch)
  • Código simples (212 linhas)
  • Sem filtros de status
  • Sem avatares
  • Sem totalizadores
{
  "type": "presence-list",
  "id": "lista_presenca",
  "dataSource": "cooperados",
  "groupField": "nome_galpao",
  "hideUntilGroup": true
}
presence_compact

PresenceCompactField

Campo avançado com avatares, filtros dropdown e totalizadores em tempo real. Ideal para listas grandes e dashboards que precisam de recursos visuais.

  • Avatares personalizados
  • Dropdown de filtros por status
  • Totalizadores visuais
  • Layout em 1 ou 2 colunas
  • Alternância rápida (clique no card)
  • Ajuste automático de altura
  • Indicadores animados
{
  "type": "presence_compact",
  "id": "participantes",
  "dataSource": "cooperados",
  "groupField": "nome_galpao",
  "hideUntilGroup": true,
  "columns": 1,
  "defaultStatus": "desligado"
}
