🚀 Sistema de Performance - Exemplos Práticos

Guia rápido de uso do sistema de otimização de performance do EcoForms

1️⃣ Uso Básico - Renderização Simples

Exemplo mais simples para começar a usar o sistema.

import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';

// Container onde o formulário será renderizado
const container = document.getElementById('my-form');

// Criar renderer com configuração básica
const renderer = new ProgressiveFormRenderer(container, {
    strategy: 'critical-first',  // Campos críticos primeiro
    chunkSize: 20,               // 20 campos por chunk
    enableMemoization: true      // Ativar cache
});

// Seus campos do formulário
const fields = [
    { id: 'nome', type: 'text', label: 'Nome', required: true },
    { id: 'email', type: 'email', label: 'Email', required: true },
    { id: 'telefone', type: 'tel', label: 'Telefone', required: false },
    // ... mais campos
];

// Renderizar
await renderer.render(fields);

console.log('✅ Formulário renderizado!');
✅ Resultado: Formulário renderizado com otimizações automáticas e feedback visual.

2️⃣ Com Monitoramento de Performance

Adicionar PerformanceMonitor para capturar métricas.

import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';
import { PerformanceMonitor } from './js/core/PerformanceMonitor.js';

// Inicializar monitor
const monitor = new PerformanceMonitor({
    trackFPS: true,
    trackMemory: true,
    trackLongTasks: true
});

const container = document.getElementById('my-form');
const renderer = new ProgressiveFormRenderer(container, {
    strategy: 'critical-first',
    chunkSize: 20,
    enableMemoization: true
});

await renderer.render(fields);

// Obter métricas após renderização
setTimeout(() => {
    const metrics = monitor.getMetrics();
    
    console.log('📊 Métricas de Performance:');
    console.log(`- FCP: ${metrics.fcp?.toFixed(2)}ms`);
    console.log(`- TTI: ${metrics.tti?.toFixed(2)}ms`);
    console.log(`- Campos: ${metrics.totalFields}`);
    console.log(`- Tempo médio/campo: ${metrics.averageFieldTime?.toFixed(2)}ms`);
    console.log(`- FPS médio: ${metrics.averageFPS}`);
    console.log(`- Long tasks: ${metrics.longTaskCount}`);
    
    // Gerar relatório completo
    const report = monitor.generateReport();
    console.log('📄 Relatório:', report);
}, 500);
✅ Resultado: Console mostra Core Web Vitals, FPS, memória e recomendações.

3️⃣ Com Progress Bar e Feedback Visual

Mostrar progresso da renderização ao usuário.

import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';
import { eventBus } from './js/core/EventEmitter.js';

const container = document.getElementById('my-form');
const progressBar = document.getElementById('progress-bar');
const statusText = document.getElementById('status-text');

// Event listeners para progresso
eventBus.on('progressiveRenderProgress', (payload) => {
    const percent = (payload.current / payload.total) * 100;
    
    // Atualizar barra de progresso
    progressBar.style.width = `${percent}%`;
    
    // Atualizar texto
    statusText.textContent = `Renderizando... ${payload.current}/${payload.total}`;
});

eventBus.on('progressiveRenderCompleted', (payload) => {
    statusText.textContent = `✅ ${payload.totalFields} campos renderizados em ${payload.renderTime?.toFixed(2)}ms`;
    
    // Esconder progress bar
    setTimeout(() => {
        progressBar.style.opacity = '0';
    }, 1000);
});

eventBus.on('fieldRendered', (payload) => {
    console.log(`✓ Campo renderizado: ${payload.fieldId}`);
});

const renderer = new ProgressiveFormRenderer(container, {
    strategy: 'priority',
    chunkSize: 20
});

await renderer.render(fields);
✅ Resultado: Progress bar animada e status textual em tempo real.

4️⃣ Formulário Grande (500+ campos)

Configuração otimizada para formulários grandes.

import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';
import { PerformanceMonitor } from './js/core/PerformanceMonitor.js';

const monitor = new PerformanceMonitor();

// Para formulários grandes, use estratégia 'virtual'
const renderer = new ProgressiveFormRenderer(container, {
    strategy: 'virtual',         // 🎯 Melhor para 500+ campos
    chunkSize: 25,               // Chunks maiores
    enableMemoization: true,     // Cache ativado
    lazyLoadSections: true       // Lazy loading de seções
});

// Gerar 1000 campos de teste
const largeFormFields = Array.from({ length: 1000 }, (_, i) => ({
    id: `field-${i}`,
    type: i % 5 === 0 ? 'select' : 'text',
    label: `Campo ${i}`,
    required: i % 10 === 0,
    section: `section-${Math.floor(i / 250)}`  // 4 seções
}));

console.log('🚀 Renderizando 1000 campos...');
const start = performance.now();

await renderer.render(largeFormFields);

const duration = performance.now() - start;
const metrics = monitor.getMetrics();

console.log(`✅ Renderizado em ${duration.toFixed(2)}ms`);
console.log(`📊 FPS médio: ${metrics.averageFPS}`);
console.log(`💾 Memória: ${metrics.currentMemory?.used}`);
console.log(`⚡ Long tasks: ${metrics.longTaskCount}`);
✅ Resultado: 1000 campos renderizados em ~3s com FPS > 50 e zero long tasks.

5️⃣ Integração com Alpine.js

Usar o sistema com Alpine.js para reatividade.

<!-- HTML -->
<div x-data="formRenderer" x-init="init()">
    <div class="mb-4">
        <progress :value="progress" max="100" class="w-full"></progress>
        <p x-text="statusText"></p>
    </div>
    
    <div id="form-container"></div>
    
    <button @click="loadForm()" class="btn">Carregar Formulário</button>
</div>

<script type="module">
import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';
import { PerformanceMonitor } from './js/core/PerformanceMonitor.js';
import { eventBus } from './js/core/EventEmitter.js';

Alpine.data('formRenderer', () => ({
    renderer: null,
    monitor: null,
    progress: 0,
    statusText: 'Pronto',
    metrics: {},
    
    init() {
        // Inicializar monitor
        this.monitor = new PerformanceMonitor({
            trackFPS: true,
            trackMemory: true
        });
        
        // Inicializar renderer
        this.renderer = new ProgressiveFormRenderer(this.$el.querySelector('#form-container'), {
            strategy: 'critical-first',
            chunkSize: 20,
            enableMemoization: true
        });
        
        // Event listeners
        eventBus.on('progressiveRenderProgress', (p) => {
            this.progress = (p.current / p.total) * 100;
            this.statusText = `Renderizando... ${p.current}/${p.total}`;
        });
        
        eventBus.on('progressiveRenderCompleted', (p) => {
            this.statusText = `✅ Completo! ${p.totalFields} campos`;
            this.metrics = this.monitor.getMetrics();
            
            console.log('Métricas:', this.metrics);
        });
    },
    
    async loadForm() {
        const fields = await fetch('/api/form-config.json').then(r => r.json());
        await this.renderer.render(fields);
    }
}));
</script>
✅ Resultado: Integração reativa com Alpine.js, progress bar e métricas no estado.

6️⃣ Adaptive Rendering (Progressive Enhancement)

Adaptar estratégia baseado no dispositivo e tamanho do formulário.

import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';

// Detectar capacidade do dispositivo
const isLowEnd = navigator.hardwareConcurrency < 4;
const isMobile = /Android|iPhone|iPad/i.test(navigator.userAgent);

// Função para escolher estratégia ideal
function getOptimalConfig(fieldCount) {
    if (fieldCount < 100) {
        return {
            strategy: 'sequential',
            chunkSize: 10,
            enableMemoization: false
        };
    }
    
    if (fieldCount < 500) {
        return {
            strategy: isLowEnd ? 'virtual' : 'critical-first',
            chunkSize: isLowEnd ? 15 : 20,
            enableMemoization: !isLowEnd
        };
    }
    
    // 500+ campos
    return {
        strategy: 'virtual',
        chunkSize: isLowEnd ? 20 : 25,
        enableMemoization: !isLowEnd,
        lazyLoadSections: true
    };
}

// Usar configuração adaptativa
const config = getOptimalConfig(fields.length);

console.log(`📱 Dispositivo: ${isMobile ? 'Mobile' : 'Desktop'}`);
console.log(`💻 CPU cores: ${navigator.hardwareConcurrency}`);
console.log(`⚙️ Configuração: ${config.strategy}, chunk ${config.chunkSize}`);

const renderer = new ProgressiveFormRenderer(container, config);
await renderer.render(fields);
✅ Resultado: Performance otimizada automaticamente para o dispositivo do usuário.

7️⃣ Benchmark Custom

Criar seus próprios benchmarks personalizados.

import { ProgressiveFormRenderer } from './js/core/ProgressiveFormRenderer.js';
import { PerformanceMonitor } from './js/core/PerformanceMonitor.js';

async function benchmarkStrategy(strategy, fields) {
    const monitor = new PerformanceMonitor();
    const container = document.createElement('div');
    document.body.appendChild(container);
    
    const renderer = new ProgressiveFormRenderer(container, {
        strategy,
        chunkSize: 20
    });
    
    const start = performance.now();
    await renderer.render(fields);
    const duration = performance.now() - start;
    
    const metrics = monitor.getMetrics();
    
    // Cleanup
    renderer.destroy();
    monitor.destroy();
    document.body.removeChild(container);
    
    return {
        strategy,
        duration: duration.toFixed(2),
        avgFieldTime: metrics.averageFieldTime?.toFixed(2),
        longTasks: metrics.longTaskCount,
        fps: metrics.averageFPS,
        memory: metrics.currentMemory?.used
    };
}

// Comparar estratégias
const fields = generateTestFields(500);
const strategies = ['sequential', 'critical-first', 'priority', 'virtual'];
const results = [];

for (const strategy of strategies) {
    const result = await benchmarkStrategy(strategy, fields);
    results.push(result);
    
    // Aguardar limpeza
    await new Promise(resolve => setTimeout(resolve, 200));
}

console.table(results);

// Exportar para CSV
const csv = results.map(r => Object.values(r).join(',')).join('\n');
console.log('\nCSV Export:\n', csv);
✅ Resultado: Tabela comparativa de todas as estratégias com exportação CSV.
